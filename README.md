# Instalando-MySql
repo documentando instalación



## Requisitos

- Sistema operativo: Windows 10/11
- Conexión a Internet
- Instalador de MySQL Installer

## Paso 1. Descargar MySQL

Ingresar al sitio oficial de MySQL y descargar el instalador correspondiente para Windows.

## Paso 2. Ejecutar el instalador

Abrir el archivo descargado y aceptar los permisos de administrador para iniciar la instalación.

**Captura 2**

## Paso 3. Configuración del servidor

Configurar el servidor MySQL:

- Puerto: **3306**
- Método de autenticación: **Strong Password Encryption**
- Crear la contraseña para el usuario **root**.
